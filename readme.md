# 한국외대 이중전공 모의지원 서비스
## 프로젝트 소개
한국외대에서 이중전공을 지원하고자 하는 이들에게, 모의지원 서비스를 제공함으로 그들의 합격 가능성을 예측하는데 도움을 주고자 한다.

## 주요 기능

**-SW**

| 기능 | 설명 |
| ------ | ------ |
| 로그인 | 유저가 접속 후 자신의 아이디(학번)과 비밀번호를 입력하여 로그인함. |
| 회원가입 | 아직 회원 가입이 되지 않은 유저는 회원가입 필수. |
| 데이터 비교 | 유저가 모의 지원한 학과에서 본인의 GPA가 몇 등인지 보여줌. |
